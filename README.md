# user controls
Player 1:
To move - arrow keys

Player 2:
To move - WASD

Player 3:
To move - keypad(8, 4, 2, 6)

# Settings
SPACE to start

R to restart

X to exit

# How to Play:
Collect the coins

Avoid the vodka - you lose a point if you hit it and slow down (once you hit 5 vodkas, you will stop moving)

Player with most points wins
